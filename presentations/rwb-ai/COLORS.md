# RWB Color Palette

Эти цвета извлечены из оригинальной презентации `pres.pptx`.

## Основные цвета

### Фиолетовые
- `#1E1753` — **Темно-фиолетовый** (основной текст, 533 использования)
- `#6B42FB` — **Яркий фиолетовый** (акценты, кнопки)
- `#3A18B5` — **Средний фиолетовый** (ссылки, hover)

### Светлые фоны (фиолетовые)
- `#F9F1FF` — **Очень светлый фиолетовый** (фон карточек)
- `#EFD5FF` — **Светло-фиолетовый** (заголовки карточек)
- `#EFD9FF` — **Светло-фиолетовый альт**
- `#B4A0FF` — **Средне-светлый фиолетовый**

### Розово-фиолетовые
- `#E9C3FF` — **Розово-фиолетовый светлый**
- `#E5B8FE` — **Розово-фиолетовый светлее**
- `#F3E2FF` — **Розово-фиолетовый самый светлый**
- `#E2B1FF` — **Розово-фиолетовый**

### Голубые акценты
- `#C5E0F9` — **Светло-голубой** (таблетки/pills)
- `#EAF4FF` — **Очень светлый голубой**
- `#D6EBFF` — **Голубой для кнопок**

### Нейтральные
- `#888888` — **Серый** (вспомогательный текст)
- `#FFFFFF` — **Белый** (фон, текст на темном)

## Использование в CSS

```css
:root {
  /* Primary */
  --rwb-purple-dark: #1E1753;
  --rwb-purple: #6B42FB;
  --rwb-purple-medium: #3A18B5;
  
  /* Light backgrounds */
  --rwb-purple-lightest: #F9F1FF;
  --rwb-purple-light: #EFD5FF;
  --rwb-purple-light-alt: #EFD9FF;
  --rwb-purple-medium-light: #B4A0FF;
  
  /* Pink-purple */
  --rwb-pink-light: #E9C3FF;
  --rwb-pink-lighter: #E5B8FE;
  --rwb-pink-lightest: #F3E2FF;
  
  /* Blue accents */
  --rwb-blue-light: #C5E0F9;
  --rwb-blue-lightest: #EAF4FF;
  --rwb-blue-pill: #D6EBFF;
  
  /* Neutrals */
  --rwb-text: #1E1753;
  --rwb-text-secondary: #888888;
  --rwb-white: #FFFFFF;
}
```

## Типографика

### Шрифты
- **Inter** (400, 500, 600, 700)

### Размеры
- **48pt** — Большие заголовки (h1)
- **28pt** — Средние заголовки (h2)
- **16pt** — Подзаголовки (h3)
- **14pt** — Малые заголовки (h4)
- **12pt** — Основной текст
- **11pt** — Мелкий текст
- **9pt** — Вспомогательный текст
- **8pt** — Примечания

### Насыщенность
- **400** — Regular (основной текст)
- **500** — Medium (заголовки)
- **600** — Semi-bold (акценты)
