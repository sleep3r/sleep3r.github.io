# RWB AI Presentation

Презентация "Оценка эффекта: бизнес-кейс, экономическая модель, риски, комплаенс"

## Доступ

После деплоя будет доступна по адресу:
`https://sleep3r.github.io/presentations/rwb-ai/`

## Локальный запуск

Просто откройте `index.html` в браузере или используйте:

```bash
# Из корня репозитория
python3 -m http.server 8000
# Затем откройте http://localhost:8000/presentations/rwb-ai/
```
